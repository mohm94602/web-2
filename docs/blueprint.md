# **App Name**: SocialSaver

## Core Features:

- URL Input: Allow users to paste the URL of a social media video.
- Video Detection: Automatically detect the video source and available download qualities using a tool to extract metadata.
- Quality Selection: Enable users to select the desired video quality and format.
- Download Initiation: Provide a button or link to start the video download.
- Download Management: Implement a download manager or integrate with the browser's default download functionality.

## Style Guidelines:

- Primary color: A vibrant sky blue (#42A5F5) to represent trust and ease of use.
- Background color: A very light blue (#E3F2FD) to provide a clean and airy feel.
- Accent color: A deeper blue (#1E88E5) to draw attention to key actions.
- Body and headline font: 'PT Sans', a humanist sans-serif, will be used for all text because of its warmth and modernity.
- Use simple, clear icons for video sources, quality options, and download actions.
- A clean and intuitive layout with a prominent URL input field and clear download options.
- Subtle animations on button hover and during the download process.
